# node_vue_waste

## Project setup
```
cd node_vue_waste

npm init -y
```


### 进入到admin目录

```
cd admin

npm install 

npm run serve
```

### 进入到web目录
```
cd web_pc

npm install 

npm run serve
```
### 进入到server目录

```
cd server

npm install 

npm run serve
```
